Chapter 1 - From GA to Hull

For digital construction of the ship in vessel.js it is necessary to introduce the offset table of the ship in a JSON file. In some cases the table can be unknown or the designer wants to start from a more visual perspective. On those cases the table of offsets must be constructed by ship line plan.
