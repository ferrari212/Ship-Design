Introduction

This web report aims to be a introduction for the best practices in using vessel.js library for a open collaborative ship project. The report will introduce the steps from the general arrangement and 3D model analysis to a final simulation of the ship in a web platform. The report will be an important source for standardize collaborative ship projects between NTNU and its several institution partners.
